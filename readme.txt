Test task for CodeIT.

To check application copy it to your server's web dir an navigate to index.php in root directory.
Then set DB settings in includes/settings.php file.
After it open application in browser, go to init DB page.
Click "Create test tables", then "Fill tables with test data". Each "Fill tables with test data" adds 20000 pseudo-random records to DB. For generating used dictionary from includes/words.txt file.
Go to index. You can test search now.
